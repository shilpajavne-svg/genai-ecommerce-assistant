# GenAI E-Commerce Shopping Assistant

An intelligent, terminal-based AI shopping assistant built in Python using the Google Gemini Pro API. It helps users discover products, filters recommendations by budget, and gives real-time personalized shopping summaries.

## Features
- **GenAI Powered:** Uses Google Gemini API for deep product understanding and human-like interactions.
- **Budget Filtering:** Adapts results precisely to user-defined budget limitations.
- **Structured Outputs:** Automatically formats responses with clear choices, pricing, and pros/cons.

## Setup Instructions

1. Install the required Google AI library:
   ```bash
   pip install -r requirements.txt
   ```

2. Get a free API Key from Google AI Studio and replace `YOUR_API_KEY_HERE` in the `ecommerce_ai.py` file.

3. Run the application:
   ```bash
   python ecommerce_ai.py
   ```
